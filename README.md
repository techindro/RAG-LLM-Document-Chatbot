# Document Chatbot

A chatbot that reads your PDF files and answers questions based on their content.

---

## Overview

This application allows you to interact with your documents through natural language questions. You upload PDF files, and the system processes them to create a searchable index. You can then ask questions, and the chatbot will find relevant information from your documents.

---

## Features

- Supports PDF and text files
- Provides answers based on document content
- Maintains conversation history
- Works on desktop and mobile devices
- Free to use and open source

---

## Requirements

Before starting, ensure you have:

- Python 3.9 or higher installed
- A free Hugging Face API key
- PDF files to test the system

---

## Getting an API Key

1. Visit huggingface.co
2. Create a free account
3. Navigate to Settings → Access Tokens
4. Click "New token"
5. Provide a name and select "Read" permissions
6. Copy the generated token
